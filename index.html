Getting Started

To begin using the WiFi Deauther:

  1. Connect to the "Dev Rafi Deauther" access point
  2. Open this web interface (192.168.4.1)
  3. Navigate to the Scan tab to discover nearby networks
  4. Select target networks (only those you have permission to test)
  5. Use the Attack tab to perform security tests

Basic Commands

Scan Networks: Discovers all nearby WiFi networks

Deauth Attack: Tests network resistance to deauthentication packets

Beacon Flood: Creates fake networks to test client behavior

Packet Monitor: Captures and analyzes WiFi packets

Safety Guidelines

  • Only test networks you own or have permission to test
  • Disable attacks immediately after testing
  • Use in controlled environments only
  • Be aware of local laws and regulations
